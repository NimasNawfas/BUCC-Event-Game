# 🎮 FIFA Face-Off – Emoji to Icon Football Quiz

An interactive, fast-paced web game created for the **BUCC (BRAC University Computer Club)** event. Test your football knowledge by decoding emoji clues and tackling trivia questions to climb the fair leaderboard!

🚀 **[Click Here to Play the Game Live!](https://nimasnawfas.github.io/BUCC-Event-Game/)**

---

## 🕹️ How to Play

1. **Decode the Emoji:** You will be presented with a creative emoji clue representing a current football star (e.g., `🐢💨` for Mbappé).
2. **Use Your Hints:** Stuck? You can reveal up to three helpful hints: **League**, **Position**, and **Country** before guessing.
3. **Face the Trivia:** Once the player's identity is revealed, you must answer **5 increasingly difficult trivia questions** about their career records, goals, and achievements.
4. **Earn Points & Compete:** Every correct trivia answer gives you 1 point. 

---

## ✨ Features

* **No Typing Required:** The modern, responsive UI is entirely button-based—perfect for a fast-paced club fair environment.
* **Persistent Leaderboard:** Scores are saved directly in local browser storage so multiple fair participants can compete for the top spot throughout the day.
* **Responsive Design:** Optimized to play perfectly on both mobile phones and desktop screens.

---

## 🛠️ Tech Stack

This project is built purely using lightweight web technologies for instant loading and performance:

* **HTML5** – Structured gameplay interface
* **CSS3** – Modern, clean layout and responsive design
* **JavaScript (ES6)** – Game logic, state management, and LocalStorage integration
